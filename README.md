# Java Slot Machine 🎰

A simple console-based slot machine game built using Java.

## Features
- Random slot symbol generation
- Betting system
- Balance tracking
- Payout multipliers
- Console-based gameplay

## Technologies
- Java
- Arrays
- Random class
- Conditional logic

## How to Run

1. Compile the program
javac Array.java

2. Run the program
java Array
